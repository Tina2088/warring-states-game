# 《策问山河》SGame2 美术资源生成提示词

## 使用说明

1. 在 ChatGPT 官网（或任何图像生成工具）里，逐张复制**「提示词」**段落发送
2. 生成后按**「文件名」**保存
3. 所有文件放到 `D:\tinaproject\SGame2\public\assets\<分类目录>\` 下
4. 生成完后告诉 Claude，它会把代码里的 `.svg` 路径改成 `.png`

---

## 六、生成建议

### 推荐生成顺序（按视觉优先级）

1. **先生成 1 张商鞅**（5.1）— 校准风格基调
2. 风格满意后继续 **其余 11 张士人** — 这是游戏里最显眼的资源
3. **4 张背景** — 氛围定调
4. **10 张数值图标** — 保持高度一致性，可尝试同一次对话里连续生成
5. **3 张国家族徽**
6. **5 张地图节点图标**

### 保持一致性的技巧

- **同一对话窗口内连续生成同类图标**：模型会记住风格
- 数值图标和地图节点这类小图 — 生成时可以说："**保持和上一张完全相同的底盘风格和配色，只更换中央汉字**"
- 士人头像 — 生成完一张后，说："**保持完全相同的构图、配色、装饰边框风格，更换人物为{下一个人物描述}**"

### 如果某张不满意

- **先尝试微调提示词**：加入具体视觉词（如"更昏暗"、"更粗的勾边"、"羊皮纸纹理更明显"）
- **再尝试重新生成**（ChatGPT 的图像生成有随机性）
- **避免大改风格**：保持整体统一优先

### 保存规范

- 文件名严格按本文档标注（全小写，带 `.png`）
- 放到对应目录（backgrounds / icons / map / advisors）
- 生成完之后告诉 Claude，它会把代码里的 `.svg` 路径改成 `.png`

# 《策问山河》SGame2 v2 新增美术资源生成提示词

## 使用说明

1. 在 ChatGPT 官网逐张复制**「提示词」**发送
2. 生成后按**「文件名」**保存
3. 放到 `D:\tinaproject\SGame2\public\assets\<分类目录>\`
4. 若生成可选资源，需要告诉 Claude 改代码引用这些图

### 必做（花 10 分钟）
**只生成 P0 的 3 张士人头像**。这是唯一不做会破图的资源。完成后告诉 Claude："士人头像已放到 advisors/ 目录"，无需改代码（代码已用 `${id}.png` 模板）。

**P2 传奇卡插画**：仅影响 5 张稀有卡的视觉表现，对整体影响小。

**P3 典藏阁背景 + 太史公头像**：纯装饰，有更好没有也无所谓。

## 保存规范

- 文件名严格按本文档标注（全小写，带 `.png`）
- 放到对应目录：
  - P0 → `public/assets/advisors/`
  - P1 → `public/assets/endings/`（新目录，需手动创建）
  - P2 → `public/assets/cards/`（新目录，需手动创建）
  - P3 → `public/assets/backgrounds/` 和 `public/assets/ui/`
- 生成完告诉 Claude，会帮你接入代码（如果需要）

---

## 保持一致性的技巧

- **同组图在同一对话里生成**：3 个士人 → 一个对话；12 个结局 → 一个对话（或按国家分 3 个对话）
- 生成第二张时说："**保持完全相同的构图、配色、装饰边框风格**，场景换成 {下一个描述}"
- 如果某国结局风格漂移，开新对话重做该国 4 张
